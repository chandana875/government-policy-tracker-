# Blockchain-Based Government Policy Tracking System

## 📖 Overview
This project is a blockchain-based application designed to ensure transparency in government policy updates. It maintains version history, timestamps, and approval tracking using Ethereum smart contracts.

## 🚀 Features
- Policy version tracking
- Immutable records
- Timestamp and approval details
- Search by version
- Compare policy updates

## 🛠 Tech Stack
- Solidity
- Ethereum (Sepolia Testnet)
- React (Vite)
- Ethers.js
- MetaMask

## 📂 Smart Contract
- addPolicy()
- getPolicy()
- changeAdmin()

## 🔗 How to Run
1. Install dependencies:
   npm install

2. Start frontend:
   npm run dev

3. Connect MetaMask
